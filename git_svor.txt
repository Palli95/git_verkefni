Git spuringar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd   -notað til að skipta um möppu
	cd ~ -fara i heimamöppu users
	ls   -List  information
	pwd  -print working directory
	mkdir-búa til möppur

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone	-afrita
	git log		-Show commit logs
	git status	-Show the working tree status
	git diff	-Show changes between commits, commit and working tree, etc
	git checkout	-Switch branches or restore working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 			-Show changes between commits, commit and working tree, etc
	b) git diff --staged 		-Shows the changes between the index and the HEAD(which is the last commit on this branch). This shows what has been added to the index and staged for a commit.
	c) git diff commit1 commit2

4. 	Hvað er útgáfustýring (e.version control)? -Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later.
